# E-Commerce Platform

A modern, full-stack e-commerce platform built with Next.js, TypeScript, and MongoDB.

## Features

*   **Product Management:** Add, edit, and delete products.
*   **User Authentication:** Secure user registration and login.
*   **Shopping Cart:** A persistent shopping cart.
*   **Stripe Integration:** Secure payments with Stripe.
*   **Search and Filter:** Easily find products with search and filtering.

## Tech Stack

*   **Frontend:**
    *   [Next.js](https://nextjs.org/)
    *   [React](https://reactjs.org/)
    *   [TypeScript](https://www.typescriptlang.org/)
    *   [Tailwind CSS](https://tailwindcss.com/)
    *   [Shadcn UI](https://ui.shadcn.com/)
*   **Backend:**
    *   [Next.js API Routes](https://nextjs.org/docs/api-routes/introduction)
    *   [MongoDB](https://www.mongodb.com/)
    *   [Mongoose](https://mongoosejs.com/)
*   **Authentication:**
    *   [NextAuth.js](https://next-auth.js.org/)
*   **Payment:**
    *   [Stripe](https://stripe.com/)

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
